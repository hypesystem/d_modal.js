Documentation

The following is the extensive documentation of d_modal. For a quick overview, see the demo.

API

1. Create a new modal with given content (text):

$.d_modal(content [, options]);

2. Turn an existing DOM element (element) into a modal:

$(element).d_modal([options]);

content

Any valid html. Will be contained in the modal that is created.

element

Any valid DOM element. Will be turned into a modal (the class .d-modal will be added to it.

options

  • blocking (boolean): If true, the modal blocks all other interaction with the page by stretching a div.d-modal-blackness over the page, until the modal is dismissed.
  • dismissable (boolean): If true, the modal may be dismissed. A button for this purpose is added to the modal.

Special Cases

The following section contains any and all documentation of irregular behaviour or special use cases that require your attention!

Dismissal of Modal

Always dismiss the modal by triggering the dismiss.d_modal event, as such: $(".d-modal").trigger("dismiss.d_modal");

Setting text and html content

Calling either text() or html() to set the content of a modal will overwrite control elements used, and may have unforeseen consequences.
